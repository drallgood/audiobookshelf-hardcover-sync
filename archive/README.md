# Archived Code

This directory contains legacy code that is no longer part of the main build. It is kept for reference and historical purposes.

## Contents

- `cmd/legacy/`: Contains the old command-line interface and related code.
- `internal/legacy/`: Contains internal legacy code that has been replaced by newer implementations.

## Building

This code is not included in the main build. To build it, navigate to the respective directory and use `go build`.

## Notes

- The code in this directory may not compile with the current version of the codebase.
- It is not maintained and is provided as-is.
- For new development, refer to the main codebase in the root directory.
